# API Gestión de Pedidos - ECOP

API REST para la gestión de pedidos, productos, clientes y categorías.

## Tecnologías Utilizadas

- **.NET 8**
- **Entity Framework Core 8.0.24**
- **SQL Server**
- **JWT Authentication**
- **AutoMapper 12.0.0**
- **FluentValidation 12.1.1**
- **Swagger/OpenAPI**
